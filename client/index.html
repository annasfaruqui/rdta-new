Features

Everything you need to know about reporting anonymously

Complete Anonymity

Users can have complete anonymity while reporting the illict use of drugs, the people involved in it, and it's details

No need to be present physically to file a report

There is no need to be physically present to file a report of drug trafficking. Users can report such cases in the comfort of their homes

Help in declining drug trafficking cases

Reporting any illicit drug trafficking activity will automatically help in declining the number of drug trafficking cases

Operations

How we operate with the help of anonymous reports

Registration of the report in the nearest police station

The anonymous report is evaluated and a case is registered in the nearest police station of the area where the drug trafficking is allegedly reported.

The matter of the report is investigated

The anonymous report is investigated by using any existing criminal data and any previous records of criminal activity in the reported areas, and also by investigating any suspects and evidences that arise from the registered report.

Commencement of Anti-Drug Trafficking Operation

After all evaluations and investigations, A police raid operation will be conducted at the reported area of the drug trafficking case
